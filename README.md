# Terraform GCP

> Terraform version should be >=1.1.0

### Run : 

```
Terraform init
```
```
Terraform plan
```
```
Terraform apply
```
### terraform.tfvars variable declaration :

- bucket_name    = "xxxxxxxxxxxxx"
- roles          = "READER"
- entity         = "allUsers"
- backend_bucket = "xxxxxxxxxxxxx"
